# Track You Calendar

A minimal, terminal-style productivity calendar with real-time cloud sync using Firebase.

## Features

- Calendar-based daily task tracking
- Firebase Authentication (Email/Password)
- Firestore cloud sync (per-user, per-day)
- Offline-first behavior (localStorage-first with merge on reconnect)
- Minimal terminal UI for fast daily use

## Tech Stack

- HTML, CSS, JavaScript
- Firebase Authentication
- Firestore
- Netlify (deployment)

## How to Run Locally

From the project root, run:

```bash
npx serve .
```

> Note: `firebase.js` must contain your Firebase project configuration for cloud sync to work.
